# 项目概述

Tool Kit 是一个基于 Tauri v2 + React + TypeScript 的跨平台开发者工具箱桌面应用。

## 技术栈

- **后端**: Rust + Tauri v2
- **前端**: React 19 + TypeScript + Tailwind CSS v4
- **数据库**: SQLite (rusqlite)
- **剪切板**: arboard (跨平台) + Win32 API (WM_CLIPBOARDUPDATE 事件驱动)
- **OCR**: Windows.Media.Ocr (原生 WinRT) / Vision (Swift)
- **构建**: Vite 8

## 全局快捷键

- `Ctrl+Shift+V`: 显示/隐藏主窗口 (全局)

## 托盘行为

- 左键单击 / 双击: 切换窗口显示/隐藏
- 右键菜单: "显示窗口" | "退出"
- 关闭窗口时最小化到托盘（默认开启）

## 崩溃诊断

应用启动时设置 panic hook，崩溃信息写入 `%LOCALAPPDATA%/tool-kit/crash.log`。
