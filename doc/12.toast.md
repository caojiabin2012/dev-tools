# 全局 Toast 提示

## 功能说明

应用内统一的复制/操作成功提示，右下角弹出，2 秒自动消失。

## 文件

- `src/lib/toast.tsx` - `notify()` / `ToastContainer` / `useToast`
- `src/App.tsx` - 挂载 `<ToastContainer />`

## 使用方式

```typescript
import { notify } from '@/lib/toast'

await copyToClipboard(text)
notify('已复制时间戳')
```

## 已接入模块 (v0.4.0+)

| 模块 | 提示示例 |
|------|----------|
| 编码工具 · 时间戳 | 已复制时间戳 / 已复制日期时间 |
| 身份证工具 | 已复制身份证号 / 已复制全部 N 条 |
| 生成器 | 已复制 UUID / 已复制密码 / 已复制全部 … |

样式：`bg-primary text-primary-foreground`，与主题一致。

## 二次开发

- 新工具复制反馈：调用 `notify(message)` 即可
- 调整时长：修改 `toast.tsx` 中 `setTimeout` 2000ms
