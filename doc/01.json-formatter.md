# JSON 格式化

## 功能说明

提供 JSON 数据的格式化、压缩和可视化树形展示功能。

## 文件

- `src/components/json-formatter.tsx` - 主组件
- `src/components/json-tree.tsx` - 树形展示组件
- `src/lib/json-themes.ts` - 主题配置

## 实现原理

1. **格式化**: 使用 `JSON.stringify(data, null, 2)` 将 JSON 字符串格式化为缩进 2 空格的可读格式
2. **压缩**: 使用 `JSON.stringify(data)` 将 JSON 压缩为单行
3. **树形展示**: 递归解析 JSON 对象，生成可折叠的树形结构
4. **错误处理**: 使用 `JSON.parse` 捕获语法错误，显示友好的错误信息
