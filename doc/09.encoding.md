# 编码转换

## 功能说明

Base64、URL 编解码，以及 Unix 时间戳与日期时间互转。

## 文件

- `src/components/encoding.tsx` - 编码工具 UI（含 `TimestampTool`）
- `src/lib/date-time.ts` - 上海时区日期格式化/解析
- `src/lib/toast.ts` - 复制成功提示

## 时间戳转换 (v0.4.0)

### 输出格式

固定为 **`YYYY-MM-DD HH:mm:ss`**（Asia/Shanghai），例如：

```
时间戳：1782464062
日期时间：2026-06-26 16:54:22
```

### 实现

```typescript
// 格式化
formatDateTimeShanghai(date)  // sv-SE locale + Asia/Shanghai

// 解析输入
parseDateTimeShanghai('2026-06-26 16:54:22')  // 按 +08:00 理解
```

### 复制行为

- 时间戳、日期时间均提供「复制」按钮
- 使用 `copyToClipboard`（Tauri 命令，比 navigator.clipboard 更稳定）
- 复制成功后右下角 Toast：`已复制时间戳` / `已复制日期时间`

### 单位切换

- **秒 (s)** / **毫秒 (ms)** 可切换
- 「获取当前时间」同时填充时间戳与格式化的日期时间（上海时区一致）
