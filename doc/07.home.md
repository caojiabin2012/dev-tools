# 首页

## 功能说明

应用启动默认进入首页，提供工具搜索、常用工具快捷入口和全部工具网格。

## 文件

- `src/components/home.tsx` - 首页 UI
- `src/App.tsx` - 默认 `activeTool: 'home'`

## 界面结构

1. **搜索框**：按工具名称、分组名称、功能提示关键词过滤
2. **常用工具**：JSON 格式化、剪切板、计算器、开发工具（4 项大卡片）
3. **全部工具**：按分组展示所有工具入口

## 工具提示文案 (TOOL_HINTS)

| 工具 | 提示 |
|------|------|
| JSON 格式化 | 格式化 · 压缩 · 校验 |
| 剪切板 | 历史记录 · 快速粘贴 |
| 计算器 | 计算 · 单位换算 |
| 日历 | 公历 · 农历 · 节假日 |
| 编码工具 | Base64 · URL · 时间戳 |
| 生成工具 | UUID · 密码 · Hash |
| 身份证工具 | 解析 · 校验 · 生成 |
| 开发工具 | 正则 · Cron |
