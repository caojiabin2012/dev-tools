# 设置

## 功能说明

管理应用配置：开机自启动、关闭时最小化到托盘、检查更新。

## 文件

- `src-tauri/src/settings.rs` - 后端设置逻辑 + 自动更新检查
- `src/lib/settings-api.ts` - 前端 API
- `src/components/settings.tsx` - 设置界面
- `src/components/about.tsx` - 关于界面

## 配置存储

配置文件位于系统数据目录：
- Windows: `%LOCALAPPDATA%/tool-kit/settings.json`
- macOS: `~/Library/Application Support/tool-kit/settings.json`
- Linux: `~/.local/share/tool-kit/settings.json`

```json
{
  "auto_start": false,
  "close_to_tray": true
}
```

## 实现原理

### 数据结构

```rust
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AppSettings {
    pub auto_start: bool,      // 开机自启
    pub close_to_tray: bool,   // 关闭到托盘（默认 true）
}
```

### 加载与保存

```rust
impl AppSettings {
    pub fn load() -> Self {
        // 1. 读取 JSON 文件，解析失败则返回默认值
        let mut settings = std::fs::read_to_string(&path)
            .and_then(|s| serde_json::from_str(&s))
            .unwrap_or_default();

        // 2. 同步 auto_start 与 OS 实际状态
        //    防止用户在系统层面修改了自启动配置后不同步
        if let Ok(enabled) = query_autostart_enabled() {
            settings.auto_start = enabled;
        }

        settings
    }

    pub fn save(&self) -> Result<(), String> {
        let json = serde_json::to_string_pretty(self)?;
        std::fs::write(path, json)?;
        Ok(())
    }
}
```

### 开机自启

使用 `auto-launch` crate，跨平台实现：

```rust
fn build_autostart() -> Result<AutoLaunch, String> {
    let exe_path = std::env::current_exe()?;
    AutoLaunchBuilder::new()
        .set_app_name("Tool Kit")
        .set_app_path(exe_path.to_str()?)
        .build()
}

fn query_autostart_enabled() -> Result<bool, String> {
    let autolaunch = build_autostart()?;
    autolaunch.is_enabled()  // 查询 OS 实际状态
}

fn update_autostart(enabled: bool) -> Result<(), String> {
    let autolaunch = build_autostart()?;
    if enabled {
        autolaunch.enable()?;
    } else if autolaunch.is_enabled().unwrap_or(false) {
        autolaunch.disable()?;  // 仅在已启用时才禁用
    }
    Ok(())
}
```

- **Windows**: 写入注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
- **macOS**: 创建 `~/Library/LaunchAgents/` plist 文件
- **Linux**: 创建 `~/.config/autostart/` .desktop 文件

`get_settings` 命令每次调用都会同步 `auto_start` 与 OS 实际状态，确保 UI 显示一致。

### 关闭到托盘

在 `lib.rs` 的 `setup` 中注册窗口事件：

```rust
main_window.on_window_event(move |event| {
    if let WindowEvent::CloseRequested { api, .. } = event {
        let settings = settings_for_tray.lock().unwrap();
        if settings.close_to_tray {
            api.prevent_close();       // 阻止关闭
            hide_main_window(&window); // 隐藏 + 从任务栏移除
        }
    }
});
```

### 窗口显示/隐藏辅助函数

```rust
fn show_main_window(window: &WebviewWindow) {
    let _ = window.set_skip_taskbar(false);  // 恢复任务栏图标
    let _ = window.unminimize();
    let _ = window.show();
    let _ = window.set_focus();
}

fn hide_main_window(window: &WebviewWindow) {
    let _ = window.hide();
    let _ = window.set_skip_taskbar(true);   // 从任务栏移除
}

fn toggle_main_window(app: &AppHandle) {
    if let Some(window) = app.get_webview_window("main") {
        if window.is_visible().unwrap_or(false) {
            hide_main_window(&window);
        } else {
            show_main_window(&window);
        }
    }
}
```

### 检查更新

通过 GitHub Releases API 获取最新版本：

```rust
pub async fn check_for_update() -> Result<Option<UpdateInfo>, String> {
    let client = reqwest::Client::new();
    let resp = client
        .get("https://api.github.com/repos/nicepkg/tool-kit/releases/latest")
        .header("User-Agent", "tool-kit-app")
        .header("Accept", "application/vnd.github+json")
        .send()
        .await?;

    let release: GitHubRelease = resp.json().await?;
    let current = env!("CARGO_PKG_VERSION").to_string();
    let latest = release.tag_name.trim_start_matches('v').to_string();

    if latest != current {
        // 查找对应平台安装包: .msi / -setup.exe / .dmg / .AppImage
        let download_url = release.assets.iter()
            .find(|a| a.name.ends_with(".msi")
                || a.name.ends_with("-setup.exe")
                || a.name.ends_with(".dmg")
                || a.name.ends_with(".AppImage"))
            .map(|a| a.browser_download_url.clone());

        Ok(Some(UpdateInfo { current_version, latest_version, download_url, release_notes }))
    } else {
        Ok(None)
    }
}
```

## API 命令

| 命令 | 参数 | 返回 | 说明 |
|------|------|------|------|
| `get_settings` | - | `AppSettings` | 获取当前设置 (同步 OS 自启动状态) |
| `save_settings` | `auto_start: bool, close_to_tray: bool` | `AppSettings` | 保存设置并应用 |
| `get_app_version` | - | `String` | 获取版本号 |
| `check_for_update` | - | `Option<UpdateInfo>` | 检查是否有新版本 |

## 二次开发

- **添加新设置项**: 在 `AppSettings` 结构体添加字段，在 `save_settings` 命令添加参数
- **添加设置页面 UI**: 在 `settings.tsx` 添加新的 `SettingItem`
- **持久化方式**: 可替换为 SQLite 或 tauri-plugin-store
- **集成自动更新**: 使用 `tauri-plugin-updater` 实现静默下载安装
