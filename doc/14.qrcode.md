# 二维码工具

## 功能说明

上传图片解析二维码，或将文本/URL 生成 PNG 二维码；历史记录持久化到本地 SQLite，支持时间筛选、搜索，当前预览与列表筛选互不影响。

## 文件

### 前端

- `src/components/qrcode/index.tsx` — 解析 / 生成双 Tab、历史列表、筛选栏
- `src/lib/qrcode-api.ts` — Tauri 命令封装

### 后端

- `src-tauri/src/qrcode/mod.rs` — 模块入口
- `src-tauri/src/qrcode/database.rs` — SQLite 读写与筛选
- `src-tauri/src/qrcode/commands.rs` — 解析、生成、历史 CRUD、复制/保存 PNG

## 数据库设计

数据库文件：`{数据目录}/qrcode.db`（与 `clipboard.db` 同级）

```sql
CREATE TABLE qr_generate_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    content_text TEXT NOT NULL,
    png_image BLOB NOT NULL,
    created_at DATETIME DEFAULT (datetime('now', 'localtime'))
);

CREATE TABLE qr_decode_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    content_text TEXT NOT NULL,
    file_name TEXT NOT NULL,
    created_at DATETIME DEFAULT (datetime('now', 'localtime'))
);
```

| 表 | 保留上限 | 说明 |
|----|----------|------|
| `qr_generate_items` | 30 条 | 文本 + PNG BLOB |
| `qr_decode_items` | 50 条 | 解析文本 + 源文件名 |

超出上限时按 `created_at` 删除最旧记录。

## 解析流程

1. 用户上传或拖入图片（PNG/JPEG/GIF/WebP/BMP，≤ 12MB）
2. 后端 `rqrr` 检测并解码二维码网格
3. 成功则写入 `qr_decode_items`，返回记录 ID
4. 前端按 ID 独立加载详情，**筛选历史时不重置当前预览**

## 生成流程

1. 用户输入文本（≤ 2048 字符）
2. 后端 `qrcode` crate 渲染 PNG（280–512px，含 quiet zone）
3. 写入 `qr_generate_items`，支持复制到剪贴板、另存为文件

## 历史筛选

与剪切板一致的时间维度：

| `time_filter` | 含义 |
|---------------|------|
| `all` | 全部 |
| `today` | 今天 |
| `yesterday` | 昨天 |
| `week` | 近 7 天 |
| `month` | 近 30 天 |

另支持 `search_query`：解析历史匹配内容与文件名；生成历史匹配内容文本。

列表 API 返回 `{ items, total }`；当前选中项通过 `get_qr_decode_item` / `get_qr_generate_item` 按 ID 加载，不在筛选结果中仍保持上方预览。

## API 命令

| 命令 | 说明 |
|------|------|
| `decode_qr_and_save` | 解析图片并入库 |
| `list_qr_decode_items` | 解析历史（含筛选） |
| `get_qr_decode_item` | 按 ID 获取解析记录 |
| `delete_qr_decode_item` / `clear_qr_decode_items` | 删除 / 清空解析历史 |
| `generate_qr_and_save` | 生成 PNG 并入库 |
| `list_qr_generate_items` | 生成历史（含筛选） |
| `get_qr_generate_item` | 按 ID 获取生成详情（含 PNG） |
| `copy_qr_generate_to_clipboard` | 复制 PNG 到系统剪贴板 |
| `save_qr_generate_file` | 另存 PNG 文件 |

## 依赖

- `qrcode` — 生成二维码
- `rqrr` — 解析二维码
- `image` — 图片解码（含 `jpeg` feature）
