# 关于与更新

## 功能说明

显示应用信息，检查并提示新版本更新。

## 文件

- `src-tauri/src/settings.rs` - `check_for_update` 命令
- `src/components/about.tsx` - 关于界面
- `src/lib/settings-api.ts` - API 封装

## 实现原理

### 检查更新

通过 GitHub Releases API 获取最新版本：

```rust
let resp = reqwest::get(
    "https://api.github.com/repos/nicepkg/tool-kit/releases/latest"
).await?;
```

### 版本比较

```rust
let current = env!("CARGO_PKG_VERSION").to_string();
let latest = release.tag_name.trim_start_matches('v').to_string();

if latest != current {
    // 有新版本，返回 UpdateInfo
}
```

### 下载安装

从 GitHub Release assets 中查找对应平台的安装包：
- Windows: `.msi`
- macOS: `.dmg`
- Linux: `.AppImage`

点击下载按钮打开浏览器，用户从 GitHub 下载安装包。

## API 命令

| 命令 | 返回 |
|------|------|
| `check_for_update` | `Option<UpdateInfo>` |

## UpdateInfo 结构

```rust
pub struct UpdateInfo {
    pub current_version: String,
    pub latest_version: String,
    pub download_url: Option<String>,      // 安装包下载链接
    pub release_notes: Option<String>,     // 更新日志
}
```
