# 日历

## 功能说明

月视图日历，支持农历、黄历、备忘录、提醒；v0.4.0 起统一主题配色，适配明暗模式。

## 文件

- `src/components/calendar.tsx` - 日历 UI
- `src/lib/lunar.ts` - 农历、节气、黄历数据
- `src/lib/calendar-store.ts` - 备忘录/提醒 localStorage 存储

## v0.4.0 配色优化

| 元素 | 样式 |
|------|------|
| 今天 | `bg-primary text-primary-foreground`，主题色高亮 |
| 选中（非今天） | `bg-primary/10 ring-primary/40` |
| 周末列标题 | `text-primary/80` |
| 节日 | `text-primary` |
| 节气 | `text-emerald-600 dark:text-emerald-400` |
| 宜/忌标签 | 半透明底色 + 边框（emerald / red） |
| 备忘/提醒圆点 | primary / amber |

此前使用硬编码 `bg-blue-500`、固定 hex 色，暗色模式下不协调；现已与全局 CSS 变量统一。

## 主要功能

- 月视图、月份切换、「今天」快捷跳转
- 农历日期、节气、节日显示
- 黄历：宜忌、干支、纳音、冲煞、星宿、吉神凶煞
- 备忘录：7 色标记，localStorage 持久化
- 提醒：支持每天/每周/每月/每年重复（UI 已就绪）
