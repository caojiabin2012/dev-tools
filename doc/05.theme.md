# 暗黑模式

## 功能说明

支持明亮、暗黑、跟随系统三种主题模式。

## 文件

- `src/lib/use-theme.ts` - 主题 Hook
- `src/index.css` - CSS 变量定义
- `src/components/sidebar.tsx` - 主题切换按钮

## 实现原理

### CSS 变量方案

在 `index.css` 中定义两套颜色变量：

```css
/* 默认 (明亮) */
:root {
  --color-background: #ffffff;
  --color-foreground: #0a0a0a;
  /* ... */
}

/* 暗黑模式 */
.dark {
  --color-background: #0a0a0a;
  --color-foreground: #fafafa;
  /* ... */
}
```

### 主题切换逻辑

```typescript
function applyTheme(theme: Theme) {
  const resolved = theme === 'system' ? getSystemTheme() : theme;
  document.documentElement.classList.toggle('dark', resolved === 'dark');
}
```

1. 在 `<html>` 元素上切换 `.dark` 类
2. 通过 CSS 变量自动应用对应颜色
3. 跟随系统: 监听 `prefers-color-scheme` 媒体查询

### 持久化

主题选择存储在 `localStorage`，键为 `dev-tools-theme`（兼容旧键 `tool-kit-theme`）。

## 二次开发

- 修改颜色: 编辑 `index.css` 中的 CSS 变量
- 添加主题: 扩展 `Theme` 类型和 `applyTheme` 逻辑
- 动画过渡: 在 `index.css` 添加 `transition: background-color 0.3s`
