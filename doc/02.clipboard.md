# 剪切板管理

## 功能说明

自动监控系统剪切板，保存文本和图片的复制历史，支持搜索、筛选、置顶、复制回剪切板、图片 OCR。

## 数据库设计

```sql
CREATE TABLE clipboard_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    content_type TEXT NOT NULL,        -- 'text' | 'image'
    content_text TEXT,                  -- 文本内容
    content_image BLOB,                -- 图片 PNG 二进制
    image_width INTEGER,               -- 图片宽度
    image_height INTEGER,              -- 图片高度
    mime_type TEXT,                     -- MIME 类型 (image/png)
    file_path TEXT,                     -- 文件路径 (预留)
    file_name TEXT,                     -- 文件名 (预留)
    file_size INTEGER,                 -- 文件大小 (预留)
    is_pinned BOOLEAN DEFAULT 0,       -- 是否置顶
    created_at DATETIME DEFAULT (datetime('now', 'localtime')),
    updated_at DATETIME DEFAULT (datetime('now', 'localtime'))
);

CREATE INDEX idx_content_type ON clipboard_items(content_type);
CREATE INDEX idx_created_at ON clipboard_items(created_at);
CREATE INDEX idx_is_pinned ON clipboard_items(is_pinned);
```

## 实现原理

### 全局状态 (mod.rs)

```rust
// 上次剪切板内容的哈希，用于去重
pub static LAST_CLIPBOARD_HASH: Lazy<Arc<Mutex<String>>> = Lazy::new(|| Arc::new(Mutex::new(String::new())));

// 标记：下一次检测到图片时跳过（防止写回时重复捕获）
pub static SKIP_NEXT_IMAGE: AtomicBool = AtomicBool::new(false);
```

### 剪切板监控 (monitor.rs)

**Windows: 事件驱动 (WM_CLIPBOARDUPDATE)**

Windows 使用 Win32 API 注册剪切板监听器，剪切板变化时系统发送 `WM_CLIPBOARDUPDATE` 消息，无需轮询：

```rust
// 1. 创建隐藏消息窗口 (HWND_MESSAGE)
let hwnd = CreateWindowExW(..., Some(HWND_MESSAGE), ...);

// 2. 注册剪切板监听
AddClipboardFormatListener(hwnd);

// 3. 消息循环处理 WM_CLIPBOARDUPDATE
while GetMessageW(&mut msg, None, 0, 0).as_bool() {
    DispatchMessageW(&msg);
}

// 4. 窗口过程回调
unsafe extern "system" fn wnd_proc(...) -> LRESULT {
    match msg {
        WM_CLIPBOARDUPDATE => {
            read_and_save_clipboard(&shared.db);
        }
        _ => DefWindowProcW(...)
    }
}
```

**非 Windows: 轮询模式**

```rust
std::thread::spawn(move || {
    while *running.lock().unwrap() {
        read_and_save_clipboard(&db);
        std::thread::sleep(Duration::from_millis(500));
    }
});
```

### 内容读取与去重 (read_and_save_clipboard)

```rust
fn read_and_save_clipboard(db: &Arc<Mutex<Database>>) -> Result<(), String> {
    let mut clipboard = arboard::Clipboard::new()?;

    // 1. 优先检测图片
    match clipboard.get_image() {
        Ok(image_data) => {
            // 跳过标记：写回剪切板时设置，防止重复捕获
            if SKIP_NEXT_IMAGE.swap(false, Ordering::SeqCst) {
                LAST_CLIPBOARD_HASH = hash(image_data);
                return Ok(());
            }

            let hash = hash_bytes(&image_data.bytes);
            if hash != *LAST_CLIPBOARD_HASH {
                // BGRA -> RGBA 字节序转换
                let rgba_bytes = convert_bgra_to_rgba(&image_data.bytes);

                // 编码为 PNG
                let png_buf = encode_to_png(width, height, rgba_bytes);

                // 存入数据库
                db.add_image_item(&png_buf, width, height, "image/png");
            }
        }
        Err(ClipboardOccupied) => return Ok(()),  // 被占用时跳过
        Err(ContentNotAvailable) => {}             // 无内容
        Err(_) => {}
    }

    // 2. 检测文本
    match clipboard.get_text() {
        Ok(text) => {
            let normalized = text.trim().to_string();
            if normalized.is_empty() { return Ok(()); }

            let hash = hash_text(&normalized);
            if hash != *LAST_CLIPBOARD_HASH {
                if !db.is_duplicate_text(&normalized) {
                    db.add_text_item(&normalized);
                }
            }
        }
        ...
    }
}
```

### 去重策略

- **文本**: SHA256 哈希 + 数据库内容查重
- **图片**: SHA256 哈希（基于原始字节）
- **写回防循环**: `SKIP_NEXT_IMAGE` 原子标记 + `LAST_CLIPBOARD_HASH` 哈希更新

### 图片写回剪切板

```rust
// copy_image_to_clipboard 命令
SKIP_NEXT_IMAGE.store(true, Ordering::SeqCst);  // 设置跳过标记

// 从数据库读取 PNG -> 解码 -> RGBA
let img = image::load_from_memory(&image_data)?;
let rgba = img.to_rgba8();

// 更新哈希（防止监控重复捕获）
LAST_CLIPBOARD_HASH = hash(&rgba_bytes);

// 写入剪切板
clipboard.set_image(ImageData { width, height, bytes: rgba_bytes })?;
```

## API 命令

| 命令 | 参数 | 返回 | 说明 |
|------|------|------|------|
| `get_clipboard_items` | `GetItemsParams` | `GetItemsResult` | 获取列表 (分页+筛选) |
| `get_clipboard_item_detail` | `id: i64` | `Option<ClipboardItem>` | 获取详情 (含图片) |
| `delete_clipboard_item` | `id: i64` | `()` | 删除条目 |
| `toggle_pin_item` | `id: i64` | `bool` | 切换置顶 |
| `clear_clipboard_history` | - | `()` | 清空非置顶记录 |
| `copy_to_clipboard` | `text: String` | `()` | 复制文本到剪切板 |
| `paste_from_clipboard` | - | `String` | 读取剪切板文本 |
| `copy_image_to_clipboard` | `item_id: i64` | `()` | 复制图片到剪切板 |
| `open_file` | `file_path: String` | `()` | 用系统默认程序打开文件 |
| `open_file_containing_folder` | `file_path: String` | `()` | 打开文件所在目录 |
| `ocr_image` | `image_data: Vec<u8>` | `OcrResult` | 图片文字识别 (含坐标) |

### GetItemsParams

```rust
struct GetItemsParams {
```
