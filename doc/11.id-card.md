# 身份证工具

## 功能说明

18 位身份证号码批量生成与单条解析。

## 文件

- `src/components/id-card.tsx` - UI
- `src/lib/id-card.ts` - 生成/解析/校验码算法
- `src/lib/region-codes.ts` - 省市区划代码
- `src/lib/toast.ts` - 复制成功提示

## 批量生成

- 地区搜索选择（省/市/区三级）
- 出生年份范围、性别（随机/男/女）、数量 1–100
- 结果表格：身份证号、地区、出生日期、性别、年龄

## 号码解析

- 输入 18 位号码，解析归属地、生日、性别、年龄
- 校验码正确性指示（绿/红圆点）

## 复制 (v0.4.0)

| 操作 | 行为 |
|------|------|
| 单条复制 | Toast：`已复制身份证号` |
| 全部复制 | 换行拼接，`已复制全部 N 条` |
| 解析页复制 | 同上 |

实现：`copyToClipboard` + `notify()`，替代 `navigator.clipboard`。

## 二次开发
