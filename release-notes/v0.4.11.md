## v0.4.11 更新说明

### 生活常识 📖

- 新增「生活常识」工具：行政区划、车牌代号、邮政编码、二十四节气、日历
- 支持车牌 / 邮编 / 地名三种搜索，点击结果自动选中省市区
- 各省车牌简称一览可点击跳转；补全河北多地市区县及雄安新区（冀X）
- 原独立「日历」入口合并至此 Tab

### 剪切板加固 📋

- Windows 剪贴板读写改为单线程 worker，降低并发导致崩溃风险
- 大图入库前降采样；Windows 原生剪贴板读写路径优化
- 增强 Windows native 崩溃诊断

### 文档

- 新增 `doc/15.upgrade.md` 版本升级与发布步骤
