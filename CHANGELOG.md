# Tool Kit 更新日志

## 2026-06-25 更新

### 一、日历功能 📅

新增日历工具，包含以下功能：

#### 1. 基础日历
- 月视图日历网格，支持月份切换
- 周一为起始日
- 今天的日期蓝色高亮显示
- 点击日期查看详情

#### 2. 农历显示
- 每个日期下方显示阴历日期（灰色）
- 农历初一显示月份名称
- 节气日显示节气名称（绿色）
- 使用 `lunar-javascript` 库确保计算准确

#### 3. 黄历信息
- **宜忌**：根据值神显示当日宜做和忌做的事
- **干支**：年柱、月柱、日柱
- **纳音**：日柱纳音五行
- **彭祖百忌**：当日天干地支禁忌
- **冲煞**：冲哪个生肖、煞哪个方位
- **星宿**：二十八星宿
- **吉神凶煞**：当日吉神和凶煞

#### 4. 五行分析
- 根据日干推算五行属性
- 显示心情建议、幸运色、吉位

#### 5. 节日显示
- 农历节日（春节、中秋等）
- 公历节日（国庆、元旦等）
- 节气
- 蓝色字体显示

#### 6. 备忘录
- 按日期添加备忘
- 支持标题和内容
- 7种颜色标记
- localStorage 持久化存储
- 日历格子上显示蓝色小圆点

#### 7. 定期提醒
- 设置提醒时间
- 支持重复：每天/每周/每月/每年
- 开关控制
- 日历格子上显示橙色小圆点

#### 8. 性能优化
- 日历组件常驻，切换时不重新加载
- 使用 CSS 控制显示/隐藏

### 二、身份证工具 🪪

新增身份证号码生成与解析工具：

#### 1. 批量生成
- **地区选择**：搜索省/市/区县，自动填充区划代码
- **出生年份**：设置起止年份范围
- **性别选择**：随机/男/女
- **生成数量**：1-100条
- **结果表格**：显示身份证号、地区、出生日期、性别、年龄
- **一键复制**：支持复制单条或全部

#### 2. 号码解析
- 输入18位身份证号码
- 自动解析：
  - 归属地区
  - 区划代码
  - 出生日期
  - 性别
  - 年龄
- **校验码验证**：绿色/红色指示

#### 3. 区域代码数据库
- 覆盖全国34个省级行政区
- 包含省/市/区县三级数据
- 支持关键词搜索
- 文件：`src/lib/region-codes.ts`

#### 4. 身份证规则说明
- 前6位：行政区划代码
- 7-14位：出生日期 (YYYYMMDD)
- 15-17位：顺序码 (奇数男，偶数女)
- 第18位：校验码 (根据前17位计算)

### 四、编码转换工具 🔄

新增编码转换工具，包含以下功能：

#### 1. Base64 编解码
- 文本编码为 Base64
- Base64 解码为文本
- 支持中文等 Unicode 字符
- 一键复制结果

#### 2. URL 编解码
- URL 编码（encodeURIComponent）
- URL 解码（decodeURIComponent）
- 适用于 URL 参数处理

#### 3. 时间戳转换
- 时间戳转日期时间
- 日期时间转时间戳
- 支持秒/毫秒切换
- 一键获取当前时间戳

### 五、生成器工具 ⚡

新增生成器工具，包含以下功能：

#### 1. UUID 生成
- **v4 (随机)**：完全随机生成，最常用
- **v1 (时间戳)**：基于时间戳生成
- 支持大写/小写选项
- 支持带/不带横线
- 批量生成（1-100条）
- 一键复制单条或全部

#### 2. 密码生成
- 自定义密码长度（4-64位）
- 可选字符类型：
  - 大写字母 (A-Z)
  - 小写字母 (a-z)
  - 数字 (0-9)
  - 特殊字符 (!@#$...)
- 密码强度指示（弱/中/强）
- 批量生成（1-50条）
- 使用 crypto.getRandomValues 确保安全

### 六、开发工具 🛠️

新增开发工具，包含以下功能：

#### 1. 正则测试
- 输入正则表达式和标志
- 实时匹配高亮显示
- 匹配结果列表（内容、位置、分组）
- 常用标志说明

#### 2. 常用正则推荐
- 手机号
- 邮箱
- 身份证
- URL
- IP地址
- 日期
- 时间
- 中文
- 银行卡号
- 邮政编码
- 十六进制颜色
- HTML标签
- 点击自动填充到输入框和测试文本

#### 3. Cron 表达式解析
- 输入 5 位 Cron 表达式
- 自动生成中文描述
- 快速选择常用表达式
- 显示下次 10 次执行时间
- 格式说明：分 时 日 月 周
